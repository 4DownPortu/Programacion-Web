Practica: elementos HTML

H1:

Los elementos de encabezado implementan seis niveles de encabezado del documento, h1 es el más importante, y h6, el menos importante. Un elemento de encabezado describe brevemente el tema de la sección que presenta. 

La sintaxis es " < H1 >*Aqui va el encabezado*< /H1 >"

P:

El elemento p representa un párrafo. Se considera párrafo a un bloque de texto compuesto de una o más oraciones que se refieren a un tema en particular y que normalmente se separa de otros por un espacio en blanco.

La sintaxis es "< p >*Aqui va el parrafo*< /p >"

IMG:

El elemento img representa a una imagen, lo cual es un recurso externo que puede ser incrustado en el cuerpo de un documento. La ubicación de dicho recurso externo debe especificarse en el atributo src..

La sintaxis es "< img src="*Aqui va la direccion de la imagen*" >

A:

La etiqueta es uno de los elementos más importantes del lenguaje HTML, esta etiqueta sirve para convertir texto normal en hipertexto, es decir, sirve para crear enlaces (links). Con la etiqueta pueden realizarse enlaces hacia documentos externos de cualquier tipo, generalmente es usada para definir enlaces hacia otras páginas web pero su uso es más amplio y no necesariamente se tiene que realizar un enlace hacia un documento externo, pues la etiqueta permite realizar enlaces internos dentro de un mismo documento, es decir, enlazar un texto hacia una parte específica del mismo documento que lo incluye y de este modo poder navegar dentro del documento.

La sintaxis es "< a =" " > < /a >

Atributos

SRC:

El atributo src contiene una ruta que apunta a la imagen que quieres poner en la página, que puede ser una URL relativa o absoluta, de la misma forma que el elemento < a > contiene los valores del atributo href.

STYLE:

El atributo style permite definir propiedades de estilo a un elemento determinado.

HREF:

El atributo href indica una URL, un enlace puede apuntar a cualquier tipo de recurso al que pueda acceder el navegador.
