Tutorial para hacer tablas en HTML

Portugal Quintero Adolfo

En esta pagina veremos como hacer tablas en html la cual explicaremos algunas de los comando que se deben de utilizar para hacer tablas, elementos para darles color, grosor del marco de la tabla como poder insertar texto, entre algunas fuciones mas.

Para comenzar programando las tablas debemos de declarlas las cual escribimos style debajo del style escribimos table, th,td para declarar tabla, datos de la tabla y encabezado.

td: significado de datos de la tabla.
th: define la celda como encabezado de la tabla.
tr: Significan la fila de la tabla

Dentro del style podemos darle un tamano a los bordes de la tabla y de igual manera un color.

Los elemntos de las celdas de la tabla pueden contener texto, Imagenes,listas, otas tablas....

Para asignarle un tamano de bordes debemos de utilizar la siguiente linea de codigo:

border: 1px solid black

1px es el tamano de grosor del borde.
Solid black es el color del borde de la tabla.

Con la siguiente linea de codigo podemos asignarle un tamano a las columnas de la tabla estas pueden ser mas pequenas o mas grandes.

th style="width:50%"

Con el comando tr y style podemos darle un ancho a las filas las cuales se le daran una anchura en pixeles, a continuacion les mostraremos el codigo para realizarlo

tr style="height:300px"

Como ya sabemospodemos darle estilos a la tabla y de igual manera podemos darle un color Para darle un color a la tabla, volvemos al apartado donde declarate tu tabla y los td dentro de las llaves y agregamos el siguiente comando

background color: blueViolet

Para aplicar un estilo utilizaremos el selector con el codigo (:nth-child(even))
